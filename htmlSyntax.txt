Hypertext Markup Lanuage (HTML)

The Tag Structure:
<tagname>Sample Text</tagname>

Tag attribute:
<tagname attribute="value">Sample Text</tagname>
<tagname class id="first-element">
<tagname class id="second-element">Second element</tagname>
This allows us to identify an element in a document

HTML elements reference:
https://developer.mozilla.org/en-US/docs/Web/HTML/Element

HTML document structure:
<!DOCTYPE html>
<html>
  <head>
    <title>Document title</title>
  </head>
  <body>
    Text inside the HTML body tag
  </body>
</html>

Containers
  •<div>
    - a generic container that has no effect until styled using CSS
    - used to group content/layout so it can be easily styled
  •<span>
    - an inline container for phrasing content
    - used to group elements for styling purposes
    - can start anywhere in a line

Lists
  Unordered list: order does not matter
    <ul>
      <li>First List Item</li>
      <li>Second List Item</li>
      <li>Third List Item</li>
    </ul>
  Ordered list: order matters
    <ol>
      <li>Order Item</li>
      <li>Order Item</li>
      <li>Order Item</li>
    </ol>
  Description/definition list
    <dl>
      <dt>Term</dt>
      <dd>Definition</dd>
    </dl>

Nested List: list within list, helpful to create site menues adding CSS
<ul>
  <li>Unordered list item</li>
  <li>
    <ol>
      <li>Order list item</li>
      <li>Order list item</li>
      <li>Order list item</li>
    </ol>
  </li>
</ul>

Hyperlink
  <a href="other_file.html">Go to another html file</a>
  <a href="my_dog.jpg">Link to a dog picture</a>
  <a href="http://google.com">Link to google</a>

  Target attribute
  _self   loads URL into same browsing
  _blank  loads URL into a new browsing
  _parent loads URL into parent browsing
  _top    loads URL into top-level browsing

  <a href="http://www.github.com" target="_blank">Github that opens on new page</a>

Mailto
  <a href="naukti:username@email.com">Email me</a>

Anchor
  Using links to navigate to anchor points into a selected document parent

  Step one: First we'll create a link with the anchor href
  <a href="#news">News Section</a>

  Step two: Then we need to create the anchor by using the name attribute
  <a name="news"></a>

  Step three: Initialize link
  <a href="news.html#nhl"><NHL news section>

Images
  <img src="sample_picture.jpg">

  We can navigate to previous folders
  <img src="../img/mypicture.jpg">

  the inline element img has height and width attribute
  <img src="mypicture.jpg" height="300" width="300">

  alt attribute defines the alternative text describing the image
    - if image URL is wrong/DNE
    - not one of the suppported format
    - provide context for screen readers
  <img src="coding.jpg" alt="Developers working on a new project">

  Figure & Figure caption
    <figure>
      <img src="coding.jpg" alt="Developers working on a new project" title="Nerds">
      <figcaption>Developers working on a new project</figcaption>
    </figure>
